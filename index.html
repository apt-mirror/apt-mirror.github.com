apt-mirror

Thanks for using apt-mirror.

A small tool that provides ability to mirror any parts (or even all) of Debian and Ubuntu GNU/Linux distributions or any other apt sources which typically provided by open source developers.

On Debian or Ubuntu simply: 

$ apt-get install apt-mirror
$ nano /etc/apt/mirror.list
$ sudo apt-mirror

This can then easily be set to run via cron or continued to run manually when you would like to update the mirror.

About

apt-mirror was originally written by Dmitriy Khramtsov to create his university's debian mirror and is currently Developed and Maintained by Brandon Holtsclaw ( @imbrandon ).
